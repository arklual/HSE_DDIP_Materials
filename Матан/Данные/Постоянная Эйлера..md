```typst
*Теорема Эйлера*:
$gamma_n = 1 + 1/2 + 1/3 + ... + 1/n < ln n : lim_(n->oo) gamma_n = gamma$\
$gamma -$ постоянная Эйлера\

*Доказательство:*\
1. Докажем убывание.
$gamma_(n+1) - gamma_(n) = 1/(n+1) + ln n - ln (n+1) = 1/(n+1) + ln n/(n+1) = 1/(n+1) - ln (n+1)/n = 1/(n+1) (1 - (n+1) ln (1+1/n)) = 1/(n+1) (1 - ln(1+1/n)^(n+1))$

Рассмотрим $b_n = (1+1/n)^(n+1)$\

$b_n/b_(n+1) = ((1+1/n)^(n+1))/((1+1/(n+1))^(n+2)) = (1 + 1/(n^2 + 2n))^(n+1) dot (n+1)/(n+2) >= (1 + (n+1)/(n^2 + 2n) ) dot (n+1)/(n+2) = (n^3+2n^2+4n+1)/(n^3 + 4n^2 + 4n) > 1$

Т.е. ${b_n}$ строго убывает и сходится к e. Значит $b_n > e$.\

Тогда $1/(n+1) (1 - ln (1+1/n)^(n+1)) < 0$ --- т.е. ${gamma_n}$ убывает.\

2. Докажем ограниченность снизу\:
$exists C: gamma_n >= C$\

$gamma_n = 1 + 1/2 + ... + 1/n - ln n >= ln (1 + 1/1) + ln(1 + 1/n) + ... + ln(1+1/(n-1)) + 1/n - ln n = ln 2 - ln 1 + ln 3 - ln 2 + ... + ln n - ln (n-1) + 1/n - ln n >= 1/n - ln(1) = \ = 1/n > 0 $\
Получили, что ${gamma_n}$ убывает и ограничена снизу, значит имеет предел по теореме Вейерштрасса [связанная №1] $qed$.
```
