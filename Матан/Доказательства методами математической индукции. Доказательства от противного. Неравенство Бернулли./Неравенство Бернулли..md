$(1+x)^n >= 1+x n$\
Докажем по индукции:
1. База: $(1+x)^1 >= 1+x dot 1$ - верно
2. Переход:
Пусть верно $(1+x)^k >= 1 + x k$ \
Докажем, что $(1+x)^(k+1) >= 1 + x(k+1)$ также верно.\
$(1+x)^(k+1) = (1+x)^k dot (1+x) >= (1+x k)(1+x)=1+x k+x(k +x k)>=
1+x k+x= 1+x(k+1)$\
Переход доказан $qed$
