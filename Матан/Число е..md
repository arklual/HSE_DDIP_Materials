$lim_(n->oo) (1+1/n)^n = e$\

Докажем сходимость по теореме Вейерштрасса [связанная №1]:
1. Докажем ограниченность:\
	
	$(1+1/n)^n =$ [по Биному Ньютона - связанная №2] $=sum_(k=0)^n C_n^k (1/n)^k 1^(n-k) = 1 + 1 + 1/2! dot n(n-1) / n^2 + 1/3! dot (n(n-1)(n-2)) / n^3 + ... = 2 + 1/2! dot (1-1/n) + 1/3! dot (1 - 1/n)(1-2/n) + ... <= 2 + 1/2! + 1/3! + ... + 1/n! <= 2 + 1/(1 dot 2) + 1/(2 dot 3) + ... + 1/ n(n+1) = 2 + 1 - 1/n < 3$

2. Докажем возрастание
$a_n = 2 + 1/2! (1 - 1/n)$ +...\

$a_(n+1) = 2 + (1 - 1/(n+1) + ...$\

$(1 - 1/n) < (1 - 1/(n+1))$\

$a_(n+1) > a_n$\
\
Полуучаем: последовательность ограничена сверху и возрастает $=>$ $a_n$ имеет предел $qed$.

$lim_(n->oo) n/(root(n, n!)) = e$\

$lim_(n->oo) sum_(n=0)^(oo) 1/(n!) = e$\

$e tilde.eq 2,718$
